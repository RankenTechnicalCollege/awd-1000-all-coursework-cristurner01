Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

give the element a position of sticky and a property that tells the element when to become stuck (top, bottom, left, right).

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

give the element a relative position and either top, bottom, left, right, the shift the element somewhere on the viewport

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

give the element a position property with a value of fixed. Add a property of either top, bottom, left, or right, to put the element in the position it will be fixed in.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

give an element a position of absolute, and then specify either top, bottom, left or right to determine its position relative to the viewport.
